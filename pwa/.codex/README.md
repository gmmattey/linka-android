# Codex local — SignallQ PWA

## Branches

Branches do fluxo Codex devem começar com `codex`.

Formato preferido:

```text
codex/pwa/<sig-id>-<descricao-curta>
```

Fallback quando a ferramenta bloquear `/`:

```text
codex-pwa-<sig-id>-<descricao-curta>
```

Exemplos:

```text
codex/pwa/sig-39-setup-vite
codex/pwa/sig-43-speedtest-web
codex-pwa-sig-39-setup-vite
```

## Pull requests

PRs do fluxo Codex devem começar com:

```text
Codex PWA —
```

Exemplo:

```text
Codex PWA — adicionar onboarding da Squad Farol
```

## Regra

Mesmo usando o mesmo repositório do Android, o Codex local trabalha somente em `pwa/`, salvo autorização explícita.
