# Components

Jetpack Compose Material 3 + custom components.

Location: `signallq-android-kotlin/app/src/main/kotlin/io/signallq/app/kotlin/ui/component/`

## Custom Components (Confirmed)

- `MiniGrafico.kt` - Chart/graph visualization
- `WifiChannelGuide.kt` - Wi-Fi channel reference
- `RotatingMessageText.kt` - Animated text
- `AppBorderGlowEffect.kt` - Visual effect
- `LinkaPulseSymbol.kt` - Logo animation
- `LinkaPulseIcon.kt` - Icon variant
- `TypewriterText.kt` - Text animation
- `OrbitSymbol.kt` - SignallQ logo
- `SheetDragHandle.kt` - Bottom sheet control
- `OrbitInputArea.kt` - Input container
- `OrbitUserMessageBubble.kt` - Chat bubble (user)
- `ContextualQuestionCard.kt` - Question display
- `OrbitActionsCard.kt` - Action buttons card
- `PulseResultCard.kt` - Result display
- `SilentSpeedtestIndicator.kt` - Test status
- `OrbitInlineQuestion.kt` - Inline question
- `OrbitTechnicalResultBubble.kt` - Chat bubble (technical)
- `OrbitWelcomeState.kt` - Welcome UI
- `OrbitThinkingBubble.kt` - Loading state
- `OrbitAiMessageBubble.kt` - Chat bubble (AI)
- `LinkaIaHeader.kt` - AI header
- `DiagnosisChipsRow.kt` - Tag/chip row
- `GaugeCircular.kt` - Circular gauge
- `OrbitTopBar.kt` - Top app bar
- `AiModelFooter.kt` - Footer UI
